# Linux Privilege Escalation Lab

## Objective
Study privilege escalation techniques in controlled environments and understand common misconfigurations.

## Environment
Practice VM / Lab Environment

## Techniques Practiced

### 1. Sudo Misconfiguration
- Enumerated sudo permissions
- Identified executable privileges
- Studied escalation paths

### 2. SUID Exploitation Concepts
- Identified SUID binaries
- Analyzed execution permissions
- Practiced controlled escalation scenarios

### 3. PATH Hijacking
- Studied insecure PATH configurations
- Observed command execution behaviour
- Practiced PATH manipulation concepts

### 4. Environment Variable Concepts
- Reviewed environment variables
- Analyzed execution context and privilege impact

### 5. Cron Job Misconfiguration
- Reviewed scheduled tasks
- Studied privilege risks from insecure cron configurations

## Learning Outcome
Developed understanding of Linux privilege escalation vectors and system hardening principles.

## Mitigation Awareness
- Apply least privilege principle
- Review sudo permissions regularly
- Audit SUID binaries
- Secure PATH variables
- Restrict cron permissions
